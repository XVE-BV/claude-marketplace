# How to open a terminal

## Windows

1. Press **Windows key + R**
2. Type `cmd` and press **Enter**

Or:

1. Click the Start menu
2. Search for **Terminal** or **Command Prompt**
3. Click to open

## Mac

1. Press **Cmd + Space** to open Spotlight
2. Type `Terminal`
3. Press **Enter**

## Linux

Press **Ctrl + Alt + T** — works on most distros (Ubuntu, Debian, Mint).

Or right-click the desktop and look for **Open Terminal**.

---

## How to paste and run a command

1. Copy the command (click it, then **Ctrl+C** on Windows/Linux or **Cmd+C** on Mac)
2. Click inside the terminal window
3. Paste:
   - Windows: **Ctrl+V** or right-click → Paste
   - Mac: **Cmd+V**
   - Linux: **Ctrl+Shift+V** or right-click → Paste
4. Press **Enter** to run
