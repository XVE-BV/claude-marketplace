Claude Code — Getting Started Guide

XVE · How to work with Claude without running into limits or getting bad results

Claude can only see a limited amount of conversation at once — think of it as a whiteboard that fills up as you chat. Once it's full, Claude starts forgetting things and making more mistakes. This guide shows you how to keep the whiteboard clean so Claude stays sharp all day.

The Ideal Session Flow

The key habit: Don't wait until Claude starts making mistakes. At around 60 % full, pause — ask for a summary, clear the whiteboard, paste it back. It takes 30 seconds and Claude stays sharp.

How Full Is The Whiteboard?

Empty 20 % — still fresh 60 % — act now 95 % Full

0–20 % — Best quality

Claude is at its sharpest

  • This is the time to explain what you want
  • Claude follows instructions best here
  • Don't skip planning — it pays off

20–60 % — Working zone

Claude is doing well, stay aware

  • Normal back-and-forth works great
  • If Claude makes a mistake, go back — don't push through
  • For big tasks, let a helper handle it

60 % — Time to reset

Don't wait any longer

  • Ask Claude for a summary of what you've done
  • Type /clear to wipe the whiteboard
  • Paste the summary — you lose nothing

90 %+ — Avoid this zone

Claude forgets most things here

  • Claude starts making more mistakes
  • It forgets what you told it earlier
  • Automatic reset happens but loses most detail

Quick Command Reference

Command When to use it What it does in plain English
/context Start of every session Shows you how full the whiteboard is before you start
/rewind Claude gives a wrong answer Go back to before the mistake — the bad answer disappears and won't affect what comes next
/xve-session-handoff Whiteboard is ~60 % full Claude writes a structured summary of everything important — decisions made, what was done, what's next
/clear After the handoff summary Wipes the whiteboard completely clean. Paste your handoff summary right after and Claude is back up to speed
/compact Try to avoid Claude summarises itself but it happens when Claude is already struggling — the summary is lower quality than doing it manually
Think of it like browser tabs: /clear closes all tabs. Your handoff summary is your bookmarks. You lose the tabs but keep everything that mattered.
